# 数据结构

## 线性表

#### 单链表（from struct import LinearLinked）

| 方法                    | 说明                   |
| ----------------------- | ---------------------- |
| LinearLinked(iter=None) | 初始化链表，可迭代对象 |
| insert(value,index=0)   | 将数据插入指定位置     |
| delete(index=0)         | 删除指定节点,默认删除  |
| clear()                 | 重置为空表             |
| getValue(index)         | 获取指定节点的值       |
| setValue(index,value)   | 修改指定节点的值       |
